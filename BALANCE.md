# Balancing

This is a simple guide for balancing mission rewards, values may be changed arbitrarily.

## Rough Values

Science rewards -> Funds reward / 4000
Reputation rewards -> Funds reward / 1000

8 Units - 16000
7 Units - 14000
6 Units - 12000
5 Units - 10000
4 Units - 8500
3 Units - 7000
2 Units - 5500
1 Units - 4000

Note 1: Satellite test missions have lower reputation and increased science rewards.
Note 2: Test missions have reduced reputation rewards.

## Extra Unit Values

AWACS += 1000
UAV += 500
