## Description

This contract pack was made after the desire for a reason to fly around kerbin and use BD Armory's ordnance surfaced.
The pack was inspired by KAF and DCA but was made with a different scope in mind, it uses Contract Configurator for all of the agency's contracts.

The pack will feature many missions, locations will be spread all throught Kerbin and its moons, adding mission locations on other planets can be done upon request.

If you would like to contribute to this project contact me through the forums or through Steam.
